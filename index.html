vimrc for python indentation

Copy of following code block into your vimrc file and you’re good to go. The set mouse =a command is only for enabling mouse click on vim. You can disregard that if you want.

syntax enable 
set mouse=a
set tabstop=4
set shiftwidth=4
set expandtab
set softtabstop=4
syntax on
set number 
filetype indent on
set autoindent
